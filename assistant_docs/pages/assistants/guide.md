## Purpose (legacy)

This is the older interactive chat interface for communicating with configured AI agents. It is functionally similar to `/chat` but is no longer the primary entry point. Active development and new features land in `/chat`, not here.

## Quick guidance for users still on this page

If a user is on `/assistants` and asks "what does this page do?", explain:

- It's the older chat interface for AI agents.
- Functionality is essentially the same as the newer `/chat` page (pick an agent, type a message, see responses).
- The newer `/chat` page should be preferred — it has the current UI, the active feature work, and the up-to-date tool-call display.

If they ask how to do something specific (chat settings, exporting a conversation, attaching files, viewing tool calls), the behavior matches the current chat surface — refer to that guide. Don't invent legacy-only mechanics that don't exist.

## Migration

There is nothing the user needs to migrate. Conversations are not transferred between the two pages, but agents, knowledge, and tools are shared — every agent available on `/assistants` is available on `/chat`.

## What to avoid telling users

- Don't position this as the "main" or "primary" chat surface.
- Don't describe UI elements in detail (this guide intentionally omits them — the legacy layout may differ from current screenshots and we don't want to mislead).
- Don't suggest features that only live on `/chat` (newer settings panels, refreshed sidebar, etc.) as if they exist here.
