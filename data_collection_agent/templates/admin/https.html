Reverse proxy guidance

In this mode, Flask binds plain HTTP on the local port and trusts X-Forwarded-Proto from a single trusted upstream.

Example Caddy config (one-liner — auto handles cert renewal):

your-internal-host.local {
    reverse_proxy 127.0.0.1:5099
}

Make sure your proxy is the only thing reachable from clients — binding Flask to 127.0.0.1 only is the default.
